Silicon Synapse Lab

Research Assistant

Developing a whole-body controller for the Unitree G1 Humanoid Robot using IsaacSim and IsaacLab to achieve dynamic locomotion and object manipulation based on MPC-guided Reinforcement Learning methods. Implementing Model Predictive Control (MPC) based on Single Rigid Body Dynamics (SRBD) and Linearized Inverted Pendulum Model (LIPM) to derive ideal contact forces and positions for each foot during locomotion tasks. I will use these MPC outputs to guide the training of an RL policy in IsaacLab, which will enable the humanoid robot to walk and manipulate objects in a simulated environment. So far, I have designed a standing controller using Quadratic Programming (QP) to solve for optimal contact forces given a desired CoM position and orientation.

November 2025

Visual SLAM

EECE 5550: Mobile Robotics

Implemented an algorithm for Visual-based Simultaneous Localization and Mapping (VSLAM) using April Tag detection to building a pose graph with the GTSAM library. Intrinsic Camera Calibration was also performed and these parameters were added to the pose graph as well. I then performed Pose Graph Optimization to fit these poses together and determine the world poses of the robot and the April Tags in the environment, all from one monocular camera.

PyClops: A Balancing Robot

Personal Project

Designing and programming a balancing robot that maintains an upright position and can drive around a room autonomously. As an initial step, I implemented an simplified environment in Pygame to simulate the behavior of the robot and the effects of gravity on the system. September-December, 2023

Building and Programming an Autonomous Drone

Introduction to Robotics Project

I assembled and programmed a quadcopter, utilizing a Raspberry Pi as the flight computer. The drone was programmed using ROS within a Linux environment. I used Optical Flow data from the onboard camera to estimate velocity, and combined this information with acceleration data to estimate global position. I implemented an Unscented Kalman Filter to reduce noise from sensor readings and produce accurate state estimates. I also created and tuned a PID controller to maintain a stable position, and velocity during flight.
